# girbyx Claude Code Plugin Marketplace

Personal plugin marketplace for [Claude Code](https://claude.ai/code). Lists plugins published under the `girbyx` GitHub account.

## Add this marketplace

```
/plugin marketplace add girbyx/claude-plugins
```

## Install a plugin

```
/plugin install harvester@girbyx
```

## Available plugins

| Name | Description | Repo |
|------|-------------|------|
| harvester | Automate Harvest time tracking — capture session work, review, and sync to Harvest API | [girbyx/harvester](https://github.com/girbyx/harvester) |
